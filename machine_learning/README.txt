«ml_classification_gp.ipynb» – код решения задачи
«data.adult.csv» – файл с данными

Подробнее о ходе решения:
  - анализ и преобразование датасета и целевой переменной
  - обучение классификаторов на вещественных признаках
      - подбор оптимального значения гиперпараметров моделей
  - масштабирование признаков
      - подбор оптимального значения гиперпараметров моделей
  - добавление в модели категориальных признаков
  - поиск новых полезных признаков
      - добавление полиномов
      - снижение размерности с помощью фильтрационных методов, жадного отбора признаков, встроенных в модель методов
  - смешивание моделей
  - сравнение построенных моделей
В коде присутствует детальное объяснение хода решения и причины применения того или иного метода.
